3. **Implement consequences** jika necessary

## ğŸŒ Inclusivity Guidelines

### Language & Communication

**Do:**
- **Use** gender-neutral language (they/them, folks, everyone)
- **Avoid** cultural assumptions atau stereotypes
- **Be** patient dengan language barriers
- **Provide** translations jika possible
- **Respect** different communication styles

**Don't:**
- **Make** assumptions tentang gender, race, atau background
- **Use** slang atau jargon yang eksklusif
- **Ignore** cultural differences
- **Mock** accents atau language abilities

### Accessibility

**Do:**
- **Provide** alt text untuk images
- **Use** descriptive link text
- **Ensure** color contrast compliance
- **Include** captions untuk video content
- **Consider** screen reader compatibility

**Don't:**
- **Rely** solely pada color untuk information
- **Use** flashing content tanpa warning
- **Ignore** keyboard navigation
- **Create** content yang tidak accessible
