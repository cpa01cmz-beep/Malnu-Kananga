## ğŸ‘¤ User Account & Authentication

### ğŸ” Login & Magic Link

**Q: Bagaimana cara login ke portal sekolah?**
A: Portal menggunakan sistem "Magic Link" tanpa password:
1. Buka halaman login
2. Masukkan email sekolah Anda
3. Check email (termasuk folder spam)
4. Click link magic yang dikirim
5. Link berlaku selama 15 menit

**Q: Magic link tidak diterima, apa yang harus dilakukan?**
A: Ikuti langkah berikut:
1. Check folder Spam/Promosi di email
2. Pastikan email yang digunakan terdaftar di sistem
3. Tunggu 2 menit sebelum request ulang
4. Jika masih tidak diterima, hubungi admin sekolah

**Q: Magic link expired, bagaimana cara login?**
A: Magic link hanya berlaku 15 menit untuk keamanan:
1. Request magic link baru dari halaman login
2. Gunakan link baru sebelum 15 menit
3. Pastikan langsung click link setelah diterima

**Q: Bisa login dengan password?**
A: Tidak, portal hanya menggunakan magic link untuk keamanan:
- Tidak perlu mengingat password
- Lebih aman dari phishing
- Tidak ada risiko password theft
