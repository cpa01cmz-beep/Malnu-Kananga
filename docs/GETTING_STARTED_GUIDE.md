## ğŸ” First Time Login

### Magic Link Authentication System

MA Malnu Kananga menggunakan sistem **Magic Link** yang aman dan mudah:

**Cara Kerja:**
1. **Request Link**: Masukkan email Anda
2. **Email Delivery**: System kirim link login unik ke email Anda
3. **One-Click Login**: Klik link untuk login otomatis
4. **Secure Session**: Session aktif selama 15 menit

**Keamanan:**
- âœ… Tidak perlu mengingat password
- âœ… Link kadaluarsa setelah 15 menit
- âœ… Rate limiting untuk mencegah abuse
- âœ… IP tracking untuk security

### Step-by-Step Login

**Step 1: Buka Halaman Login**
```
https://ma-malnukananga.sch.id/login
```

**Step 2: Masukkan Email**
- Gunakan email yang terdaftar di sistem
- Format: `nama@ma-malnukananga.sch.id`
- Pastikan email aktif dan dapat diakses

**Step 3: Cek Email**
- Buka inbox email Anda
- Cari email dari "noreply@ma-malnukananga.sch.id"
- Subject: "Link Login MA Malnu Kananga"
- Jika tidak ada, cek folder spam/promotions

**Step 4: Klik Magic Link**
- Link berlaku 15 menit
- Akan langsung login ke dashboard
- Tidak perlu input password

**Step 5: Verifikasi Login**
- Anda akan melihat nama Anda di dashboard
- Session aktif selama 15 menit
- Akan otomatis logout jika tidak aktif

### Troubleshooting Login

**âŒ Link tidak diterima:**
- Cek folder spam/promotions
- Pastikan email terdaftar di sistem
- Coba lagi setelah 1 menit (rate limiting)

**âŒ Link kadaluarsa:**
- Request magic link baru
- Link hanya berlaku 15 menit
- Gunakan segera setelah diterima

**âŒ Email tidak terdaftar:**
- Hubungi administrator sekolah
- Daftar email Anda ke sistem
- Pastikan format email benar

## ğŸ”§ Troubleshooting

### Common Issues & Solutions

**âŒ Website Tidak Bisa Diakses**
- **Cause**: Internet connection atau server maintenance
- **Solution**: 
  1. Cek koneksi internet
  2. Clear browser cache
  3. Coba dengan browser lain
  4. Hubungi admin jika masih error

**âŒ Magic Link Tidak Diterima**
- **Cause**: Email salah, spam filter, rate limiting
- **Solution**:
  1. Cek folder spam/promotions
  2. Verify email address
  3. Tunggu 1 menit lalu coba lagi
  4. Hubungi admin untuk verifikasi email

**âŒ Login Gagal Terus**
- **Cause**: Browser cookies, session expired
- **Solution**:
  1. Clear browser cookies
  2. Gunakan incognito/private window
  3. Update browser ke versi terbaru
  4. Coba dengan device lain

**âŒ AI Assistant Tidak Merespons**
- **Cause**: No internet, server overload
- **Solution**:
  1. Cek koneksi internet
  2. Refresh halaman
  3. Tunggu beberapa saat lalu coba lagi
  4. Gunakan fitur lain yang tersedia

**âŒ Data Tidak Update**
- **Cause**: Cache, sync delay
- **Solution**:
  1. Pull to refresh
  2. Clear app cache
  3. Logout dan login kembali
  4. Hubungi guru untuk konfirmasi

### Performance Tips

**Untuk Loading Cepat:**
1. Gunakan WiFi atau 4G yang stabil
2. Update browser ke versi terbaru
3. Clear cache secara berkala
4. Close tabs yang tidak digunakan

**Untuk Battery Saving:**
1. Enable dark mode (jika tersedia)
2. Reduce push notification frequency
3. Use WiFi instead of 4G/5G
4. Close app saat tidak digunakan

### Browser Issues

**Chrome Issues:**
- Clear cache: `Ctrl+Shift+Del` (Windows) atau `Cmd+Shift+Del` (Mac)
- Update Chrome: Settings â†’ About Chrome
- Disable extensions yang tidak perlu

**Safari Issues:**
- Clear cache: Settings â†’ Safari â†’ Clear History
- Update iOS: Settings â†’ General â†’ Software Update
- Reset Safari settings jika perlu
