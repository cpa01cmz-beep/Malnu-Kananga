# UI Components

This directory contains the enhanced UI components for MA Malnu Kananga with mobile-first design, accessibility features, and micro-interactions.

## Recent Enhancements

### UX Improvements (2026-02-10)

- **Enhanced Mobile Interactions**: Touch-optimized components with haptic feedback
- **Bottom Sheet Patterns**: Mobile-friendly modal variants with swipe-to-close
- **Fluid Typography**: Responsive text sizing with clamp() functions  
- **Micro-interactions**: Ripple effects, press states, and smooth transitions
- **Loading Skeletons**: Customizable variants with shimmer animations
- **Accessibility**: WCAG AA compliance with reduced motion support

## Component Categories

- **Form Components**: Input, SearchInput, Button, Select, etc.
- **Layout Components**: Card, Modal, DataTable, Pagination, etc.
- **Feedback Components**: LoadingSkeleton, Alert, ProgressBar, etc.
- **Navigation Components**: Breadcrumb, Pagination, Tab, etc.

All components are built with:
- TypeScript for type safety
- Tailwind CSS for styling
- Accessibility-first approach
- Mobile-first responsive design
- Performance optimizations
